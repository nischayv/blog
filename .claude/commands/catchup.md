# Branch Catchup

Read and analyze all files that have been modified, added, or deleted in the current git branch compared to origin main.

First, identify all changed files using git, then read each modified/added file to understand the current state of changes in this branch.

In case there are no changes in Git locally, look at the last commit and compare that with origin main to understand the diff.
